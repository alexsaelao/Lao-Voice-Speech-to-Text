# Privacy Policy for Lao Voice - Speech to Text

**Last updated: July 15, 2026**

Lao Voice - Speech to Text is a Chrome extension that converts speech to text and, when the user chooses, inserts the resulting text into an editable field on the current web page.

## Data the extension handles

The extension handles only the information needed to provide its speech-to-text feature:

- **Speech and transcript text.** Chrome's built-in Web Speech API processes speech to produce a transcript. Lao Voice does not operate a speech-recognition server and does not send speech recordings or transcripts to a developer-operated server.
- **Selected editable field.** When the user selects a place to type on a web page and starts the feature, the extension temporarily accesses that field only to insert the transcript. It does not collect page content for analytics, advertising, profiling, or sale.
- **Local preferences.** The selected recognition language and the automatic-typing preference are stored locally in the browser using Chrome extension storage.

Speech recognition is provided by the browser and may be subject to the browser provider's own terms and privacy practices.

## How data is used and retained

Data is used solely to provide the speech-to-text feature requested by the user. Lao Voice does not sell, rent, share, or transfer user data to third parties for advertising, analytics, profiling, or any purpose unrelated to this feature.

The extension does not maintain a developer-operated database of speech, transcripts, web-page content, or user identities. Transcript preview text remains in the extension panel until the user clears it or closes/reloads the panel. Local preferences remain in the browser until the extension is removed or its extension storage is cleared.

## Permissions

Lao Voice uses access to the current tab only after the user opens the extension. This access is used to locate the editable field chosen by the user and insert the transcript into that field. It uses the Chrome Side Panel to show the controls and Chrome storage to save local preferences.

## Your choices

You can turn off automatic typing, clear the transcript preview, revoke microphone access in Chrome, or uninstall the extension at any time.
